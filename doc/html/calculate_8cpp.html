void calculate_frame ( Frame &  currentFrame,
Frame &  previousFrame,
Simul &  simul 
)

Do calculations on a frame for each eligible* linker.

Eligible linkers are those that are both present in the current frame and in the previous frame. Calculations include:

  • Calculate the force vector on each hand for a linker
  • Calculate the direction vector for each hand
  • Calculate the velocity vector for each hand
  • Calculate the wDot contribution from the linker
  • Add wDot contribution from all linkers in the frame to the wDot value for the trajectory
Parameters
currentFrame- reference to Frame object
previousFrame- reference to Frame object
simul- reference to Simul object, which contains data on simulation parameters

Iterate through each linker in the frame

void calculate_velocity_vector ( Simul &  simul,
Linker &  linker 
)

Calculate the velocity vector of a hand of a linker for a given frame.

First, calculate the velocity magnitude: $v = v_0 (1 + \vec{f}\cdot\vec{d}/f_0)$ Then, multiply the direction vector by the calculated magnitude.
